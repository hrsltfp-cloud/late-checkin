# 🚀 คู่มือการติดตั้งระบบ Late Check-in สำหรับแผนกบุคคล (HR)

ระบบถูกออกแบบมาให้แผนกบุคคล (HR) สามารถนำไปติดตั้งบน Google Account ของแผนกได้ด้วยตัวเองอย่างง่ายดาย โดยมีขั้นตอนดังนี้ครับ:

## ส่วนที่ 1: การตั้งค่าฝั่ง Google Sheet (สำหรับ HR)

> [!IMPORTANT]
> ขั้นตอนนี้ให้ HR ล็อกอินด้วยบัญชี Google ของแผนกบุคคล เพื่อให้ข้อมูลทั้งหมดถูกเก็บไว้ที่ส่วนกลางอย่างปลอดภัย

1. **สร้าง Google Sheet ใหม่**: 
   - ไปที่ [Google Sheets](https://sheets.google.com) สร้างแผ่นงานเปล่าตั้งชื่อว่า "บันทึกเข้างานสาย"
2. **เปิด Apps Script**: 
   - ที่เมนูด้านบน เลือก **ส่วนขยาย (Extensions)** > **Apps Script**
3. **ใส่โค้ดระบบ**: 
   - ลบโค้ดเริ่มต้นออกให้หมด
   - เปิดไฟล์ [Code.gs](file:///d:/Google%20antigravity%20Project/Late%20Checkin/Code.gs) ที่ผมสร้างให้ แล้วคัดลอกโค้ดทั้งหมดไปวาง
   - กดปุ่ม 💾 (บันทึก) หรือ Ctrl+S
4. **นำระบบขึ้นใช้งาน (Deploy)**:
   - กดปุ่มสีน้ำเงินมุมขวาบน **การทำให้ใช้งานได้ (Deploy)** > **การทำให้ใช้งานได้รายการใหม่ (New deployment)**
   - คลิกไอคอน ⚙️ (ฟันเฟือง) ตรง "เลือกประเภท" แล้วเลือก **เว็บแอป (Web app)**
   - **การตั้งค่า**:
     - *คำอธิบาย*: Check-in API
     - *ทำงานในฐานะ*: **ฉัน (บัญชี HR)**
     - *ผู้ที่มีสิทธิ์เข้าถึง*: **ทุกคน (Anyone)** (สำคัญมาก: ต้องเลือก "ทุกคน" เพื่อให้มือถือพนักงานส่งข้อมูลเข้ามาได้)
   - กด **การทำให้ใช้งานได้ (Deploy)** 
   - (Google อาจจะถามขอสิทธิ์เข้าถึงข้อมูล ให้กด Review Permissions และ Allow/อนุญาตไปตามขั้นตอน)
5. **รับ URL**:
   - เมื่อเสร็จแล้ว คุณจะได้ **"URL ของเว็บแอป"** (Web app URL) ยาวๆ ให้กด "คัดลอก" ไว้

## ส่วนที่ 2: การเชื่อมต่อเว็บสแกนเข้ากับฐานข้อมูล

> [!TIP]
> ตอนนี้เรามีฐานข้อมูลพร้อมแล้ว ขั้นต่อไปคือเอา URL ที่ได้มาเชื่อมกับหน้าตาเว็บแอปเพิ่อให้ทำงานร่วมกัน

1. เปิดไฟล์ [script.js](file:///d:/Google%20antigravity%20Project/Late%20Checkin/script.js) 
2. ค้นหาบรรทัดที่ 16:
   ```javascript
   const SCRIPT_URL = 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE';
   ```
3. ลบคำว่า `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE` และวาง **URL ของเว็บแอป** ที่คัดลอกมาจากขั้นตอนที่แล้วลงไปแทน (ระวังอย่าลบเครื่องหมาย `'` ออก)
4. กดเซฟไฟล์ `script.js`

## ส่วนที่ 3: การนำไปใช้งานจริง (สร้าง QR Code)

1. คุณสามารถนำไฟล์ [index.html](file:///d:/Google%20antigravity%20Project/Late%20Checkin/index.html), [style.css](file:///d:/Google%20antigravity%20Project/Late%20Checkin/style.css), และ [script.js](file:///d:/Google%20antigravity%20Project/Late%20Checkin/script.js) ไปอัปโหลดขึ้น Web Hosting ฟรี (เช่น GitHub Pages, Vercel) เพื่อให้พนักงานเข้าถึงได้ผ่านอินเทอร์เน็ต
2. เมื่อได้ URL ของเว็บไซต์มาแล้ว (เช่น `https://mycompany-checkin.vercel.app`) ให้นำ URL นี้ไปสร้างเป็น QR Code ผ่านเว็บไซต์สร้าง QR Code ฟรี
3. พิมพ์ QR Code ไปแปะไว้หน้าประตู เมื่อพนักงานมาสายและสแกน จะเข้าสู่หน้าระบบและกดบันทึก ข้อมูลจะวิ่งเข้า Google Sheet ของ HR ทันทีพร้อมประทับเวลาอย่างแม่นยำ!
